# 🛒 Superstore Gold Membership Campaign Analysis

## 📌 1. Project Overview

This end-to-end marketing analytics project evaluates the effectiveness of a Gold Membership promotional campaign conducted by a retail superstore chain. Using customer demographics, purchasing behavior, campaign interactions, and spending patterns, the analysis identifies high-value customer segments, measures campaign performance, and provides data-driven recommendations to improve future membership conversions and customer retention strategies.

## 💡 4. Key Performance Indicators (KPIs) & Insights

### A. Campaign Performance Summary

| KPI                                  | Performance              |
| :----------------------------------- | :----------------------- |
| Total Customers                      | 2,240                    |
| Membership Acceptance Rate           | 14.91%                   |
| Average Customer Income              | $51.69K                  |
| Total Customer Revenue               | $1.36M                   |
| Best Performing Channel              | Store Purchases (38.96%) |
| Average Recency (Accepted Customers) | 35 Days                  |

---

### B. Customer Segmentation & Behavioral Insights

* **Senior customers (70+ years)** generate the highest average spending levels, indicating strong purchasing power despite relatively lower digital engagement.
* **Widow customer segments** demonstrate the highest Gold Membership acceptance rates, making them a valuable target audience for future campaigns.
* Customers who accepted the Gold Membership offer generated approximately **36% higher web purchases** than non-responders.
* Customers with recent purchase activity show significantly higher conversion potential compared to dormant customers.

---

### C. Product & Revenue Analysis

| Evaluation Category                  | Top Performing Segment |
| :----------------------------------- | :--------------------- |
| 🏆 Highest Spending Product Category | Wine Products          |
| 🥩 Second Highest Spending Category  | Meat Products          |
| 🌐 Best Digital Conversion Segment   | Web Purchasers         |
| 💰 Highest Value Customer Group      | High-Income Customers  |

**Key Observation:** Wine and Meat products collectively contribute approximately **85% of total product spending**, highlighting premium-product customers as the most valuable membership prospects.

## 🎯 5. Statistical Modeling
* **Hypothesis Testing:** Run to mathematically prove the spending behavioral differences between campaign responders and non-responders.
* **Predictive Model:** Built a **Logistic Regression model** using variables like *Income*, *Recency*, and *Web Purchases* to predict which customers have the highest likelihood of accepting a membership offer next.
